# Başlangıç

Stackvo'a hoş geldiniz! Bu bölüm, Docker tabanlı geliştirme ortamınızı sıfırdan kurmak, yapılandırmak ve ilk projenizi oluşturmak için gereken tüm bilgileri kapsamlı bir şekilde içermektedir. Giriş seviyesinden başlayarak, sistem gereksinimlerini öğrenecek ve hızlı başlangıç adımlarıyla ilk projenizi çalıştıracaksınız.
